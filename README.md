# Password Security Tool 🔐

## Project Description
A web-based password strength checker and generator that helps users create secure passwords and evaluate their strength.

## Features
- Password strength checker
- Strong password generator
- Visual strength meter

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Run
Open index.html in any browser.
